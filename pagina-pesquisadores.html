Pesquisadores - Espécies Nativas de Chateaubriand

Acesso completo às informações detalhadas sobre as espécies nativas, com funcionalidades para cadastrar novas espécies e atualizar dados existentes.

Funcionalidades Disponíveis para Pesquisadores

Pesquisadores e cientistas têm acesso completo aos dados sobre as espécies nativas, incluindo informações científicas avançadas. Além disso, podem cadastrar novas espécies e atualizar as informações existentes com dados relevantes para a comunidade científica.

Principais Funcionalidades:

  • Cadastro de novas espécies na plataforma
  • Informações detalhadas sobre cada espécie
  • Possibilidade de atualizar informações e adicionar fontes científicas

Para realizar estas ações, é necessário fazer login com suas credenciais de pesquisa científica.
